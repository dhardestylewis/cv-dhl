# Resumes

Resumes organized by page count and cover letter status.

## Structure

```
resume/
├── 1-page-TODO/              # Empty - awaiting 1-page resume additions
├── 2-page/
│   ├── with-cover-letter/    # 2-page resumes including cover letter
│   └── without-cover-letter/ # 2-page resumes without cover letter
└── 2plus-page/
    └── with-cover-letter/    # Multi-page resumes with cover letter
```

## Hierarchy

`[page-count]/[cover-letter-status]/[resume-name]/`

Example: `2-page/with-cover-letter/resume-dhl-20240124-arity/`
